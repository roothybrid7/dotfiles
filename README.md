# Dotfile by yadm

## Getting Started

### Set Hub credential

created `.config/hub`

### Set CocoaPods trunk

```bash
pod trunk register orta@cocoapods.org 'Orta Therox' --description='macbook air'
```

### Add npm user

created `.npmrc`

```bash
nodenv install 7.4.0
nodenv rehash
nodenv shell 7.4.0
npm adduser
```
